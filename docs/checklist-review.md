# 記事チェックリスト（公開前確認） — Issue #13

誰がチェックしても同じ品質になるための標準チェック観点。**非属人化の要**。
スマホのWordPressアプリで下書きを開き、上から順に確認する。所要：1記事あたり数分。

## 🔴 必須（1つでもNGなら公開しない）

### 1. 事実確認 ―「どこの国の企業か」
- [ ] ブランドの**国籍・企業情報が明らかな誤りでない**
  - 怪しい/根拠が薄いと感じたら**公開しない**（ゴミ箱へ）
  - 判断に迷う時：ブランド名＋「どこの国」で軽く検索して矛盾がないか
- [ ] 「★信頼度評価」が極端・不自然でない

### 2. 薬機法・景表法（NG表現）
- [ ] 「治る」「効果がある」「No.1」「最高」「絶対」等の**断定・誇大表現がない**
- [ ] 医療・健康・美容に関する**効能効果をうたっていない**
- [ ] 「業界初」「世界一」等の**根拠なき最上級表現がない**

### 3. アフィリエイトリンク
- [ ] 記事内に**商品リンク（楽天カード）が表示される**（※プレビューで確認）
- [ ] リンク先の商品が**記事の商品と大きくズレていない**
- [ ] カードが「リンク」のテキストのままになっていない（描画されている）

## 🟡 推奨（直せるなら直す／致命的でなければ公開可）

### 4. タイトル・見出し
- [ ] タイトルに**ブランド名とカテゴリ**が入っている
- [ ] 見出し構成が揃っている（はじめに／企業／レビュー／比較／まとめ）
- [ ] 明らかな**誤字脱字・文字化けがない**

### 5. 本文の質
- [ ] 同じ内容の**繰り返し・破綻がない**
- [ ] 空のセクション（見出しだけで中身なし）がない
- [ ] 口コミが**不自然に偽物っぽくない**

## 判断フロー
```
事実NG・薬機法NG・リンク無し → ゴミ箱へ（公開しない）
軽微な誤字/言い回し           → その場で直して公開
すべてOK                      → 公開
```

## 運用メモ
- 同じブランドで何度も「企業情報が怪しい」が出る → 管理者に連絡し `company_hint` を整備
- 将来、🔴の一部（NG表現・リンク有無・空セクション）は自動QA（Issue #16）で機械チェックし、
  担当者は🔴1（事実確認）に集中できるようにする。
